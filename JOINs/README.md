# Joins

A JOIN is used to combine data from two or more related tables based on a common column.

## Topics Covered

- CROSS JOIN
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL JOIN
- Choosing between JOINs and Subqueries

## Learning Outcomes

After completing this section, I learned how to:

- Combine data from multiple related tables
- Use different types of JOINs
- Match rows using common columns
- Include unmatched rows with LEFT, RIGHT, and FULL JOINs
- Retrieve meaningful information from relational databases
- Choose between JOINs and subqueries based on the use case
