# GROUP BY

The `GROUP BY` clause is used to group rows with the same values and perform aggregate calculations on each group.

## Topics Covered

- GROUP BY
- Aggregate Functions (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`)
- Complex GROUP BY
- HAVING Clause

## Learning Outcomes

- Group records by one or more columns
- Perform calculations using aggregate functions
- Group data using multiple columns
- Filter grouped results with `HAVING`
- Understand the difference between `WHERE` and `HAVING`
